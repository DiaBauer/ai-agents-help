# ai-agents.help

**Kuratierte Plattform für GPT-Agenten & KI-Tools**

Dies ist der offizielle Code für die Startseite mit Leo Agenti.  
Enthält HTML, CSS und eine JSON-Datei für die Agentenstruktur. Bereit zur Integration in Vercel oder GitHub Pages.

## Inhalte

- `index.html`: Startseite mit Header, Agentenbereich und Footer
- `style.css`: Designstruktur (Dark Mode, Akzent Türkis)
- `agents.json`: Beispielagenten
